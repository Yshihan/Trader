# 测试指南

## 🧪 测试步骤

### 1. 启动调试
1. 在 VSCode 中打开 Trader 项目
2. 按 `F5` 启动调试模式
3. 会打开一个新的 VSCode 窗口（Extension Development Host）

### 2. 检查插件加载
在左侧活动栏中应该能看到"潜心学习"图标（书本图标）

### 3. 测试各模块功能

#### ✅ 热门板块
- 点击左侧图标
- 查看"热门板块"视图
- 应该显示20个板块数据
- 如果显示"半导体"、"人工智能"等，说明数据获取成功
- 如果显示空白或默认数据，查看调试控制台日志

#### ✅ 国内指数
- 查看"国内指数"视图
- 应该显示：
  - 上证指数
  - 深证成指
  - 沪深300
  - 创业板指
- 检查是否有乱码（应该都是正常的中文）

#### ✅ 全球指数
- 查看"全球指数"视图
- 应该显示：
  - 道琼斯（美国）
  - 纳斯达克（美国）
  - 标普500（美国）
  - 恒生指数（香港）
  - 日经225（日本）
  - 富时100（英国）
  - 德国DAX（德国）
  - 法国CAC40（法国）

#### ✅ 我的关注
1. **添加股票**
   - 点击"我的关注"视图标题栏的 `+` 图标
   - 输入股票代码或名称，如：
     - `600519`（茅台）
     - `茅台`
     - `贵州茅台`
   - 选择要添加的股票

2. **置顶股票**
   - 右键点击股票项
   - 选择"置顶"
   - 股票应该移动到列表顶部，前面有个星标

3. **取消置顶**
   - 右键点击已置顶的股票
   - 选择"取消置顶"

4. **删除股票**
   - 右键点击股票项
   - 选择"取消关注"

5. **清空所有**
   - 点击"我的关注"视图标题栏的垃圾桶图标
   - 确认清空

6. **刷新数据**
   - 点击任意视图标题栏的刷新图标
   - 所有数据应该更新

### 4. 查看调试日志
1. 在调试窗口中按 `Ctrl+Shift+U` 打开输出面板
2. 选择"调试控制台"
3. 查看日志输出，应该能看到：
   ```
   从新浪获取到 XX 个概念板块
   从新浪获取到 XX 个全球指数
   ```

## ⚠️ 可能的问题和解决方案

### 问题1：热门板块无数据
**原因：** 新浪板块代码格式可能不对

**解决方案：** 我会提供替代方案，改用热门股票显示

### 问题2：国内指数乱码
**原因：** GBK编码处理问题

**解决方案：** 已安装 iconv-lite，应该不会有问题

### 问题3：网络请求超时
**原因：** 网络问题或API限流

**解决方案：** 
- 检查网络连接
- 稍后重试
- 查看是否有降级数据显示
