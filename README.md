# 潜心学习 (Trader)

一个简洁的 VSCode 股票行情插件，让你在编码时也能关注市场动态（仅供学习使用）。

## ✨ 功能特性

- 🔄 **智能刷新** - 交易时间 3 秒自动刷新，非交易时间暂停
- 🔍 **智能搜索** - 支持股票代码/名称模糊搜索，单个/批量添加
- 📊 **实时行情** - A股、美股、板块、指数全覆盖
- 🎯 **自选管理** - 拖拽排序、置顶、多种排序方式
- 🎨 **简洁界面** - 原生 TreeView，无广告打扰

## 📋 四大模块

### 1. 国内指数
上证指数、深证成指、创业板指、科创50 等核心指数实时行情。

### 2. 热门板块
40+ 行业 ETF 板块数据，按实时涨跌幅排序，包含：
- 科技类：半导体、芯片、人工智能、5G、新能源等
- 消费类：白酒、医疗、医药、家电等
- 金融类：证券、银行、保险等
- 周期类：有色、稀土、钢铁、煤炭等

### 3. 我的关注
自选股票列表，支持：
- 📝 **添加股票** - 单个/批量添加，支持逗号、空格、换行分隔
- 🔄 **拖拽排序** - 自由调整股票顺序
- ⭐ **置顶功能** - 重要股票一键置顶
- 📊 **多种排序** - 按涨跌幅/价格/添加时间排序
- 🗑️ **快速删除** - 单个删除或一键清空

### 4. 美股市场
美股三大指数 + 科技七姐妹（苹果、微软、谷歌、亚马逊、英伟达、特斯拉、Meta）实时行情。

## 🚀 使用方法

### 基本操作
1. 安装插件后，点击侧边栏的"潜心学习"图标
2. 四个模块自动加载数据
3. 交易时间（9:30-11:30, 13:00-15:00）自动 3 秒刷新

### 添加股票
1. 点击"我的关注"模块右上角的 `+` 图标
2. 输入股票代码或名称（支持批量）：
   ```
   单个：600519 或 茅台
   批量：茅台, 平安银行, 000001
   ```

### 管理股票
- **拖拽排序** - 直接拖动股票项调整顺序
- **置顶** - 点击股票右侧的 ⭐ 图标
- **删除** - 点击股票右侧的 ✕ 图标
- **排序** - 点击右上角排序图标，选择排序方式
- **清空** - 点击右上角清空图标

### 刷新数据
点击任意模块右上角的刷新图标，手动刷新所有数据。

## 📦 安装与开发

```bash
# 安装依赖
npm install

# 编译
npm run compile

# 开发模式（监听文件变化）
npm run watch
```

按 `F5` 启动调试模式，或打包后安装 `.vsix` 文件。

## 🛠️ 技术栈

- **TypeScript** - 类型安全
- **VSCode Extension API** - TreeView、ThemeIcon、ThemeColor
- **Axios** - HTTP 请求
- **新浪财经 API** - 数据来源

## 📊 数据来源

本插件使用新浪财经公开 API 获取数据：
- A股实时行情
- 行业板块数据
- 国内/全球指数
- 美股市场数据
